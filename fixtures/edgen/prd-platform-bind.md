# US-EDGEN-042 查看社交平台绑定状态

**作为** 已登录的 Edgen 用户  
**我希望** 在设置页查看 Twitter 等平台的绑定状态  
**以便** 确认账号是否已连接并决定是否解绑

## 验收标准

1. 用户进入「平台绑定」页，可看到已支持平台列表（含 Twitter）。
2. 调用 `GET /v2/platform/bind/{platform}` 可返回当前用户的绑定状态（已绑定 / 未绑定）。
3. 未登录或 token 无效时返回 401，前端提示重新登录。
4. 绑定列表接口返回分页数据，单页默认不超过 50 条。
5. 响应体 `code` 为 0 表示业务成功；`data` 中含平台标识与绑定时间（若已绑定）。

## 非功能

- 接口需在 BETA / PROD 环境可访问，域名见服务端设计文档。
